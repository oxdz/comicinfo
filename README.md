# ccinf

ccinf 可用于部分网站的漫画下载，仅限学习用途

## 快速开始

从 Release 页面下载对应的包含 ccinf 或 ccinf.exe 的压缩包，解压后使用。

启动解压后的二进制文件，输入 url 后开始下载。

ccinf 将会启动 chrome 浏览器，如果网站需要登录，请在输入 url 前，在 ccinf 打开的浏览器中提前登录。

- Linux

```sh
./ccinf start
```

- Windows

```sh
./ccinf.exe start
```
